# 🎲 Rodando o Front End no Desenvolvimento (servidor)

## Instale as dependências
$ npm install

## Execute a aplicação em modo de desenvolvimento
$ npm run server

## O servidor iniciará na porta:8080 - acesse <http://localhost:8080>
